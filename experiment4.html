grid-auto-columns / grid-auto-rows

Allows us to control the size of an implicitly-created track. Explicit grids are defined in the usual way (grid-template-columns or grid-template-rows); implicit grids are created when we try and place an item outside the explict grid.

The blue divs are an explict 4/3 (row/column) grid, size 1fr in both dimensions. Clicking the button spawns a red div with coordinates randomly chosen from the range (1-10) x (1-10). If the div happens to have coordinates outside the explicit grid bounds, then its size is determined by the grid-auto-columns and grid-auto-rows properties.

        .wrapper {
                display: grid;
                grid-template-rows: repeat(4, 1fr);
                grid-template-columns: repeat(3, 1fr);
    
                grid-auto-rows: 6rem;
                grid-auto-columns: 3rem;
    
                grid-gap: 0.2rem;
            }
